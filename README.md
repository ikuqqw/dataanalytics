# E-Commerce Data Analytics Project

A project focused on analyzing e-commerce data using **Python, SQL, Pandas and Matplotlib**.

### Project Goals
- Calculate key business metrics (Revenue, AOV, retention, repeat purchase rate)
- Analyze customer segments and product categories
- Build visual reports and prepare insights

### Tech Stack
- Python (Pandas, Matplotlib, SQLite3)
- SQL
- VS Code

### Results
- Revenue analysis by product category
- Monthly sales and revenue trends
- New vs repeat customer breakdown
- Top-10 performing products

### Project Structure
- ecom.py ‚Äî main script
- outputs/ ‚Äî charts and CSV reports
- `requirements.txt ‚Äî dependencies

### Visualizations
- Revenue by Category
- Orders by Month
- Revenue by Month
- First-time vs Repeat Orders

### How to Run
pip install -r requirements.txt
python ecom.py
