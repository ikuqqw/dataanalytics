# E-Commerce Data Analytics Project

Проект по анализу данных интернет-магазина с использованием **Python, SQL, Pandas и Matplotlib**.

### Цели проекта
- Рассчитать ключевые метрики (выручка, AOV, retention, повторные покупки)
- Проанализировать клиентов и категории товаров
- Построить графики и подготовить отчет

### Стек
- Python (Pandas, Matplotlib, SQLite3)
- SQL
- VS Code

### Результаты
- Анализ выручки по категориям
- Динамика заказов по месяцам
- Доля новых vs повторных клиентов
- Top-10 продуктов

### Структура проекта
ecom.py — основной скрипт
outputs/ — графики и csv отчеты
requirements.txt — зависимости


### Визуализации
- Revenue by category
- Orders by month
- Revenue by month
- First-time vs Repeat orders

### Как запустить
- pip install -r requirements.txt
- python ecom.py
